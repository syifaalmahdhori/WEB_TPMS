Tentang Sistem TPMS Fuzzy Sugeno

Deskripsi Singkat

Sistem TPMS (Tire Pressure Monitoring System) berbasis Fuzzy Logic Metode Sugeno ini dirancang untuk memantau kondisi tekanan, suhu, dan kecepatan kendaraan secara real-time. Data dari sensor dikirim ke mikrokontroler ESP32, kemudian diproses menggunakan metode fuzzy untuk menentukan status ban: Aman, Waspada, atau Bahaya.

Tujuan utama sistem ini adalah meningkatkan keselamatan berkendara dengan memberikan informasi kondisi ban secara langsung melalui antarmuka web responsif yang dibangun menggunakan HTML, CSS (Bootstrap 5), dan JavaScript.

Metode Fuzzy Sugeno

Metode Fuzzy Sugeno digunakan karena dapat menghasilkan nilai output numerik (crisp value) yang efisien dan mudah diolah mikrokontroler. Proses melibatkan tiga tahap utama:

  1. Fuzzifikasi – mengubah input sensor (tekanan, suhu, kecepatan) menjadi derajat keanggotaan fuzzy.
  2. Inferensi – menerapkan aturan (rule base) menggunakan operator min untuk menentukan tingkat aktivasi.
  3. Defuzzifikasi – menghitung nilai crisp (Z) dengan metode weighted average.

Dengan pendekatan ini, sistem dapat mengklasifikasikan kondisi ban secara adaptif dan akurat.
